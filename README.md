Este proyecto implementa una versión simplificada del cifrado de la máquina Enigma utilizando dos microcontroladores ESP32 programados en MicroPython. Se emplea el protocolo ESP-NOW para permitir la comunicación directa entre los dispositivos sin necesidad de WiFi. Uno de los ESP32 actúa como emisor, cifra el mensaje con el algoritmo Enigma y lo envía al receptor usando su dirección MAC. El receptor descifra el mensaje utilizando la misma configuración de rotores, garantizando coherencia y simetría en el proceso. Todo el desarrollo y ejecución del código se realiza en el IDE de Thonny, que facilita la carga del firmware y la transferencia de archivos al ESP32.
